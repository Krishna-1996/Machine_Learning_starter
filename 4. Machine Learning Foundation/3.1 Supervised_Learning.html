Supervised Learning

Supervised learning is a type of machine learning where the algorithm is trained on a labeled dataset, meaning each input data point is associated with a corresponding target value. The goal of supervised learning is to learn a mapping from input variables to output variables based on the labeled training data.

In supervised learning, the algorithm learns from examples provided in the form of input-output pairs. It tries to approximate the mapping function so that when presented with new input data, it can predict the output accurately.

There are two main types of supervised learning:

  1. Regression: Regression models are used when the target variable is continuous. The goal is to predict a continuous value output based on input features. Examples include predicting house prices based on features like area, number of bedrooms, etc.
  2. Classification: Classification models are used when the target variable is categorical. The goal is to predict the class or category of the input data. Examples include spam detection, sentiment analysis, and image classification.

Linear Regression

a. Simple Linear Regression

Simple linear regression is a linear approach to modeling the relationship between a dependent variable and one independent variable.

b. Multiple Linear Regression

Multiple linear regression is an extension of simple linear regression that predicts a dependent variable using multiple independent variables.

Logistic Regression

a. Binary Logistic Regression

Binary logistic regression is used to model the probability of a binary outcome based on one or more predictor variables.

b. Multinomial Logistic Regression

Multinomial logistic regression is used to model the probability of multiple classes of outcomes based on predictor variables.
